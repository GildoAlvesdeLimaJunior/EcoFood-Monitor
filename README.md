# EcoFood Monitor

Sistema inteligente para redução do desperdício de alimentos em supermercados e restaurantes, integrando controle de validade e temperatura.

## Pages

- **`index.html`** — Login (simulado)
- **`entrada.html`** — Cadastro de lotes com scan simulado e Eco-Score
- **`dashboard.html`** — Dashboard com gráficos, tabela dinâmica e alertas

## Tech Stack

- HTML + TailwindCSS (CDN) + CSS custom properties
- Vanilla JavaScript (modular)
- Chart.js (CDN)
- localStorage para persistência
- GitHub Actions para deploy no GitHub Pages

## Deploy

Push na branch `main` faz deploy automático no GitHub Pages via GitHub Actions.
